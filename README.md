# MM Forums

## Generate express project
```
/> npm install -g express-generator nodemon
/> express --view=ejs mm-forums
/> cd mm-forums
/mm-forums/> npm install
/mm-forums/> npm start
```

Visit http://localhost:3000

## Clone from git
```
/> git clone --branch Ver.0.0.1 https://github.com/oberak/mm-forums.git
/> cd mm-forums
/mm-forums/> npm install
/mm-forums/> npm start
```
Visit http://localhost:3000
